GetFX packageÂ¶

Provides handling of FX rates using extrnal FX API.

It submits request to external FX API, parse the response and prints it in predefined manner. Each specific FX API provider requires new module based on getfx (e.g. getfx.getfxnbp iplements NBP FX API).

Modules:

  • getfx.getfx - base functionality to be extended by specific API

  • getfx.getfxnbp - specific NBP API implementation

  • getfx.cmdparser - parsing commandline interface
